# 2048-readme-engine

Engine for running 2048 on user Github profiles' README.md.

Welcome to the 2048 game on GitHub!

To play, open an issue with the title `MOVE: [DIRECTION]`, where `[DIRECTION]` can be `U`, `D`, `L`, or `R`.
